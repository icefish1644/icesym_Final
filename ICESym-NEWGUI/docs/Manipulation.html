Add, Edit and Delete objects

Creating Objects

There are three ways to create an object:

  • Component->New->[object_name].

  • Selecting its icone in the Toolbar

  • Copying for an existing object (select it, press "Ctrl+C" and "Ctrl+V"). This option will copy all the data excepting its connections and it's label.

After that, you must complete the form to set the parameters for the Simulation.

Editing Objects

There are two ways to edit an object:

  • Double click over its image in the diagram.

  • Double click over its label in the objects tree, at the left of the screen.

Deleting Objects

There is only one way to delete an object: selecting it and press the "Supr" key.
