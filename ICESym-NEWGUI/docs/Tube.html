Tube

This object models a tube in the simulation.

List of Attributes

  • Label: Choose a label for this object to identifies it in the object-tree and in Post Process.

  • Number of Nodes: Number of nodes.

  • Degrees of Freedom: Number of degrees of freedom.

  • Length: Pipe length [m].

  • Equispaced: Switch (0/1) if you want to make automatically the array xnod, dividing the length of the pipe in nnod nodes equispaced.

  • Has Curvature: Switch (0/1) if you want that the pipe has curvature.

  • Xnod: Coordinates in x for each node [m].

  • Diameter: Pipe diameter [m].

  • Twall: Wall pipe temperature [K].

  • Curvature: Curvature in each node along the pipe.

  • State: Initial State. You need to complete it only if Configuration->Get Inital State is set from Here.

  • Type Save: You can choose the way to save data (which nodes).

  • Normalized Points: You can choose how many points you want to save. After, you must specify the normalized positions of that points.
    Example: If the pipe length is 0.2m, and you needs save the data in the nodes at 0.0m, 0.1m and 0.2m, you must choose Normalized points = 3 and in the table below you must input 0 - 0.5 - 1.
