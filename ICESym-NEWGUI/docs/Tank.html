Tank

This object models a tank in the simulation.

List of Attributes

  • Label: Choose a label for this object to identifies it in the object-tree and in Post Process.

  • Number of Nodes: Number of nodes (un-editable, because is equal at the number of tubes connected + the internal node).

  • Degrees of Freedom: Number of degrees of freedom.

  • Volume: Volume of the tank [m^3].

  • Mass: Mass of gas contained into the tank [kg].

  • H film: Heat-transfer coefficient [W/m^2 K].

  • Area Wall: Tank wall area [m^2].

  • Temperature Wall: Tank wall temperature [K].

  • Cd Ports: Discharge coefficients of the ports.

  • State: Initial State. You need to complete it only if Configuration->Get Initial State is set from Here.

  • Connections and own State Box: Switch (0/1) if you want to save the data in each connection for Post Process.

  • Save Extras: Switch (0/1) if you want to save extra data for the tank.
