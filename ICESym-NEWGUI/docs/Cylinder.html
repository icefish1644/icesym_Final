Cylinder

This object models a cylinder in the simulation. If a simulation has some cylinder object, it is considered an engine simulation. The formulary is divided into six sections (different tabs). Each one allows you to set different parameters.

Cylinder Attributes

Defines the geometry principally, moreover allows set others parameters such as temperatures, scavenges, etc.

  • Label: Choose a label for this object to identifies it in object-tree and in Post Process.

  • Number of Zones: How many zones is the interior of the cylinder divided.

  • Degress of Freedom: Number of degrees of freedom.

  • Bore: Cylinder bore [m].

  • Crank Radius: Crank radius [m].

  • Vol Clearance: Minimum volume of the combustion chamber [m^3].

  • Head Chamber Area: Surface area of the wall combustion chamber [m^2].

  • Piston Area: Piston crown area [m^2].

  • Delta CA: Lag angle between crank shafts (only applicable for opposed-piston engines) [deg].

  • Rod Lenght: Connecting rod length [m].

  • Temperature Type: Homogeneus or No Homogeneus.

  • Temperatures: Cylinder temperature [ÂºK]. If is selected Homogeneus you only must set the wall temperature. Else, if is selected No Homogeneus you must specified the Piston, Intake, Exhaust and Liners Temperatures.

  • Model ht: Heat-transfer model:Annand model, Woschni model or Taylor model.

  • Factor ht: Coefficient affecting the heat-transfer law.

  • Mass C:Mass of fuel, air and residual gases into the cylinder [kg].

  • Type Ignition: Ignition type: Spark-ignition or Compression ignition.

  • Has Scavenge: Switches on/off (1/0) the scavenge computation.

  • Scavenge Type: Scavenge type: uniflow, scre, yam1, yam6, cd or qubcr.

  • SRv: Scavenge ratio by volume.

  • Full Implicit: Switches on/off (1/0) the implicit computation of the cylinder and valves updates.

  • State: Initial State. You need to complete it only if Configuration->Get Inital State From is set From Here.


Fuel Attributtes

Defines the characteristics of the used fuel.

  • Q Fuel: Fuel lower heating value [J/kg].

  • y: H/C ratio of fuel.

  • Hvap Fuel: Fuel heat of vaporization [J/kg].


Injection Attributtes

Defines the characteristics of the injection. Only is used if the engine has Spark Ignition.

  • Pulse: Shape of the injection pulse: Squared sine-wave, Square wave, User-defined (actives Mass Injection Rate).

  • Mass Injected per Cycle: Total injected mass per pulse [kg].

  • dTheta injected: Duration of the injection pulse in crank angle degrees.

  • Temperature Fuel: Fuel injection temperature [K].

  • Theta Injected Initial: Injection starting angle.

  • Ignition Delay Model: Ignition delay model:Livengood and Wu model, Hardenber and Hase model,Constant user-defined delay.

  • theta Id: Ignition delay angle (only needed for ignition delay model 2) [deg].

  • Mass Injected Rate:Two-column array with the user-defined injection pulse (angle and injected mass rate).

Combustion Attributtes

Defines the characteristics of the combustion.

  • Theta Ignition 0: Combustion starting angle.

  • dtheta Combustion: Duration of the combustion in crank angle degrees.

  • Phi: Fuel/air equivalence ratio.

  • Comb Efficiency Parameter: Combustion efficiency parameter for the Wiebe function.

  • Shape Parameter: Shape parameter for the Wiebe function.

  • Combustion Model: Combustion model for compression-ignition engines: Mass fraction burned profile defined by the user, Two Wiebe functions, Three Wiebe functions or Watson model.

  • Burned Mass fraction: Two-column array with the user-defined mass fraction burned.
