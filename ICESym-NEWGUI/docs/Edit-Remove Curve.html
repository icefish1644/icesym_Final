Remove or Edit a Curve in Post Process

ICESym-GUI Post Process allows you to edit a specific curve by just selecting it and pressing the "F" Key. You will notice that you selected the curve because it will be painted in a blue color. In the Edit Curve Dialog, you can change the color, style and width of the selected curve. To des-select a selected curve, just click again in it.

To erase a curve, once is selected, just press the "Delete" Key.
