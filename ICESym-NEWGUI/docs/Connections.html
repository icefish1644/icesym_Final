Connection Between Objects

Perhaps the most important operation to do in the Modeling is the connection between components. For an operational engine model, the first requirement is the relationship between its parts in a natural way. This natural way determines what objects can be connected with others.

Moreover, there are other restrictions that limit the connections possibilities. To connect the objects, you must select the left object (left, not means left in the screen), and right click in the right object. In the simulation, this means that the exhaust in the first object is intake in the second object. But, be careful, that some type of objects only can be connected with only one specific type of object. This system rules are based in the real engines, and can be resumed in the next table:

Object From Object To Number Intakes Number Exhaust
Tube All 1 1
Junction Tube N N
Atmosphere Tube 1 or 0 1 or 0
Cylinder Exhaust Valve N N
Intake Valve Tube (in intake) - Cylinder (in exhaust) 1 1
Exhaust Valve Cylinder (in intake) - Tube (in exhaust) 1 1
Tank Tube N N



Example: Connecting Objects

In this example, we shall to explain how connect a simulation composed for: Atmosphere-Tube-Atmosphere (for example, if we need to verify the behavior of the flux in the tube when there are differents conditions in the intake atmosphere and the exhaust atmosphere).

  • 1 - Create the three objects: two atmospheres and one tube (see the first image)


  • 2 - Select the first Atmosphere and do right click over the Tube. This action will connect the objects.

  • 3 - Select the Tube and do right click over the second Atmosphere. Now we have the connections required.


  • 4 - For remove connections, select the line that represents the connection (it will be colored in red), and press the "Supr" key.
