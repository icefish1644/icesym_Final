Import Data From Files

ICESym-GUI allows you to complete most of the tables with manual input or importing an array from ascii file. The requeried format is the standart octave export format and it MUST contains exactly the same number of values that the field to complete.

For example, to complete the values of Tube->Temperature Wall, you can choose Constant or Linear values, but if you want to use a more specific law, you can import that values using the Load button.


If Tube->nnod = 10, you must choose an ascii file with ten values and the next format:

#--------START FILE---------------#
value0
value1
value2
value3
value4
value5
value6
value7
value8
value9
#--------END FILE---------------#
